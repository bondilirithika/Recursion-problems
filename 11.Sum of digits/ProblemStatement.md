# Sum of Digits
Given a number, N. Find the sum of all the digits of N
 
# Examples:
**Example 1:**

**Input:**
N = 12

**Output:**
3
**Explanation:**
Sum of 12's digits:
1 + 2 = 3

**Example 2:**

**Input:**
N = 23

**Output**
5

**Explanation:**
Sum of 23's digits:
2 + 3 = 5

# Expected Complexities
Expected Time Complexity: O(logN)

Expected Space Complexity: O(1)
 

# Constraints:
1<=N<=10^5
