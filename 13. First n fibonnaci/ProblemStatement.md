# First n Fibonacci
Given a number N, find the first N Fibonacci numbers. The first two number of the series are 1 and 1.

# Examples:
**Example 1:**

**Input:**
N = 5

**Output:** 1 1 2 3 5

**Example 2:**

**Input:**
N = 7

**Output:** 1 1 2 3 5 8 13

# Expected Complexity
Expected Time Complexity: O(N).

Expected Auxiliary Space: O(N).

# Constraints:
1<= N <=84
