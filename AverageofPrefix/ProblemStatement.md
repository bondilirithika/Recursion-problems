# Average of Prefix:

Given an array arr, find the average or mean of the prefix array at every index.

**EXAMPLE-1:**

Input: arr[] = [10, 20, 30, 40, 50]

Output: [10, 15, 20, 25, 30] 

Explanation: 10 / 1 = 10, (10 + 20) / 2 = 15, (10 + 20 + 30) / 3 = 20 and so on.

**EXAMPLE-2:**

Input: arr[] = [12, 2]

Output: [12, 7]
