# Reverse a String

You are given a string s. You need to reverse the string.

# Examples:

**Input:**
s = Geeks

**Output**: skeeG

**Example 2:**

**Input:**
s = for

**Output:** rof

# Complexities

Expected Time Complexity: O(|S|).

Expected Auxiliary Space: O(1).

# Constraints:
1 <= |s| <= 10000
