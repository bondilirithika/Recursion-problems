# Factorial
Given a positive integer, N. Find the factorial of N.
 
# Examples:
**Example 1:**

**Input:**
N = 5
**Output:**
120
**Explanation:**
5*4*3*2*1 = 120
**Example 2:**

**Input:**
N = 4
**Output:**
24
**Explanation:**
4*3*2*1 = 24


# Expected complexity:
Expected Time Complexity: O(N)

Expected Space Complexity: O(1)
 

# Constraints:
0 <= N <= 18
